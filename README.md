# Análise de Dados Climáticos de Araras e Percepção da Sociedade - Extensionista IV
Projeto elaborado durante a disciplina Atividade Extensionista IV do curso de Engenharia da Computação na UNINTER 

## Objetivos do Projeto:

- Coletar dados climáticos históricos da cidade de Araras;
- Processar os dados utilizando a linguagem de programação Python;
- Analisar tendências de temperatura e precipitação ao longo do tempo;
- Desenvolver visualizações gráficas simples e acessíveis;
- Elaborar e aplicar um formulário digital para coleta da percepção da população;
- Comparar a percepção popular com os dados analisados;
- Divulgar os resultados de forma acessível;

## Metodologia
Os dados analisados foram extraídos do site [NASA POWER](https://power.larc.nasa.gov/data-access-viewer/), configurando os parâmetros conforme abaixo:

•	Single Point:
-	Data Resolution = "Standard Resolution"
-	User Community = "Renewable Eneregy"
-	Temporal Level = "Daily"
-	Location = "Latitude -22.36 | Longitude -47.38"
-	Time Extent = "1/1/1981” – “Data Atual"
-	Parameters = "Preciptation" + “Temperature at 2 Meters” + “Temperature at 2 Meters Maximum” + “Temperature at 2 Meters Minimum” 
-	(Advance Parameters) Time Standard = "UTC"
-	Data Download = "CSV"

Todo o projeto foi realizado utilizando ferramentas do Google, onde o código Python foi gerado via Google Colab, o Formulário Digital foi elaborado via Google Forms e os slides de apresentação via Google Slides
